# java-cli-gradle-spring-findbugs-pmd-cucumber-mockito-testng-car-data

## Description
A POC for a spring app using testng,
pmd, cucumber and mockito framework.

## Tech stack
- java
- gradle
	- spring
  - testng
  - cucumber
	- mockito
  - pmd
  - findbugs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
